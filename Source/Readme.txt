Классы, их предназначения:
DamageTaker - родитель пешки игрока и врагов, сделан для обработки урона, в нём добавил виджет статуса здоровья
DialogWidget - виджет по типу "да, нет". В нём лежат два ивента для подвязки на нажатие кнопок.
GameStructs - все структуры и перечисления, используемые в игре.
HPBarWidget - виджет полоски здоровья
MainMenuHud, TankHud - худы, один для главного меню. другой для самой игры.
MainMenuWidget - самый страшный класс :D виджет главного меню со всем функционалом. Есть два ивента для управления худом
TankGameModeBase - гейммод, через него я вызываю GameOver
WidgetSubsystem - "худ", который вы реализовали на уроке. Я подобный функционал видел в GameInstanceSubsystem, что и сделал
